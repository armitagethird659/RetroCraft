17. FAQ

Do I need to know how to code?

No. RetroCraft is fully visual. The HTML Snippet block is there for advanced users who want to add custom code, but everything else is drag-and-drop with visual editors.

Is my exported site really free to host?

Yes. The exported HTML/CSS files are pure static content. GitHub Pages, Netlify, and Vercel all offer free hosting for static sites. No server costs, no databases, no maintenance.

Can I use my own domain?

Yes. All three recommended hosts (GitHub Pages, Netlify, Vercel) support custom domains. Configure your domain's DNS to point to your host, then add the custom domain in your host's settings.

Will my site work on mobile?

Yes. RetroCraft generates responsive CSS with a @media (max-width: 768px) breakpoint that adjusts layouts for smaller screens.

Can I edit my site after exporting?

Yes — your project is saved in RetroCraft. Open it any time, make changes, and re-export. You can also edit the exported HTML directly if you know HTML/CSS.

What happens to my assets if I stop using RetroCraft?

Assets uploaded through RetroCraft are stored on Base44's CDN. If you stop using the platform, those URLs may stop working. For permanent hosting, download your images and re-host them on your own server or a free image host before they expire.
