6. Working with Blocks

Blocks are the building blocks (pun intended) of your pages. Each page is a vertical stack of blocks. You add, edit, reorder, and remove blocks to build your content.

Adding a Block

  • Click the [+] button between any two existing blocks to insert at that exact position
  • Or click [+ Add Block] at the bottom of the page
  • A modal opens showing all block types, organized by category:
    • Content — Text, Heading, Button, Link, Divider, HTML Snippet
    • Media — Image, Gallery, Music Player
    • Decorative — Marquee, Badge Row
    • Interactive — Email Button, Guestbook

Editing a Block

Click any block to select it. The right sidebar updates to show that block's settings. For text blocks, you can also edit text directly on the canvas by clicking into it.

Block Controls (hover overlay)

When you hover over a block, you'll see action buttons:

  • ↑ / ↓ — Move the block up or down
  • ⧉ Duplicate — Copy the block immediately below
  • 🗑 Delete — Remove the block
  • ✏️ Edit — Focus the block for editing

Resizing Blocks

When a block is selected, the right sidebar includes a Width slider (10%–100%) with presets:

  • 25% 50% 75% 100%
  • An alignment toggle lets you position narrower blocks left, center, or right
!TIP For images that are too large, use the Image block's sizing controls: Original Size, Fit Width, or Custom (set exact pixel or percentage dimensions). No need to resize in a separate app!
