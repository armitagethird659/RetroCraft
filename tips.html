16. Tips & Best Practices

Design Tips

  • Start with a theme, then customize. Pick the closest theme to your vision, then tweak colors and fonts. Don't fight the theme — work with it.
  • Use marquees sparingly. One marquee is nostalgic. Five marquees is chaos. Use them for announcements or fun messages, not body content.
  • Badge rows belong at the bottom. Classic web placement. They're great for "sites I like" or community badges.
  • Background images + tile = instant retro. A small repeating pattern GIF as a tiled background transforms the feel instantly.
  • Custom cursors add personality. But keep them 32×32px or smaller for browser compatibility.

Content Tips

  • Break content into pages. Don't put everything on one scrolling page. A Home page with links to About, Blog, and Links pages feels more authentic.
  • Use dividers between sections. They break up content visually and add retro charm.
  • Alt text on images. It's good practice for accessibility and shows up if an image fails to load.

Export Tips

  • Preview before exporting. Always check the preview to make sure everything looks right.
  • Test on mobile. RetroCraft exports responsive CSS. Open your exported site on a phone to verify it looks good.
  • Keep your original assets. If you plan to move hosts later, keep local copies of your images so you're not dependent on CDN URLs.

Image Sizing

  • Too big? Use the Image block's Custom size option to set exact pixel or percentage dimensions — no external resizing needed.
  • Cursor images must be 32×32px or smaller for cross-browser support.
  • Background patterns work best at small sizes (100×100px to 400×400px) when tiling.
  • Badges should be 88×31px (the classic web button standard) for the Badge Row block.
