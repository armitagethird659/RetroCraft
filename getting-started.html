1. Getting Started ⸜(｡˃ ᵕ ˂ )⸝♡

RetroCraft runs entirely in your web browser. No downloads, no installations — just open the app and start building.

What You'll Need

  • A modern web browser (Chrome, Firefox, Edge, or Safari)
  • Images, GIFs, or audio files you want to use (optional — built-in stickers are included)
  • A free GitHub account if you want to publish to GitHub Pages (optional)

The Big Picture

  1. Create a project and pick a retro theme
  2. Add pages (Home, About, Blog, Links, etc.)
  3. Add blocks to each page (text, images, marquees, badges, music players)
  4. Customize colors, fonts, cursors, and backgrounds
  5. Preview your site live
  6. Export as a ZIP file or publish directly to GitHub Pages
⭐!TIP RetroCraft exports pure HTML and CSS — no JavaScript frameworks, no runtime dependencies. Your exported site loads instantly and works on any browser, any host, forever.
