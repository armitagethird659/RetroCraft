13. Exporting Your Site

When you're happy with your site, export it as a downloadable ZIP file containing everything needed to host it.

What's in the ZIP

my-site-export.zip
├── index.html        ← Homepage
├── about.html        ← About page
├── blog.html         ← Blog page
├── style.css         ← All theme styles
└── assets/           ← Cursor images and bundled assets
    └── README.txt    ← Notes on externally hosted assets

How to Export

  1. Click [Export ZIP] in the top bar
  2. Wait for the compilation to finish (usually a few seconds)
  3. Your browser downloads {site-name}-export.zip

Hosting Options

The exported files are pure static HTML/CSS — they work on any web host:
Host Cost How
GitHub Pages Free Upload files to a repo, enable Pages (see next section)
Netlify Free Drag the ZIP contents onto netlify.com/drop
Vercel Free Upload via the Vercel dashboard or CLI
Any web host Varies Upload files via FTP or file manager
!NOTE Assets uploaded through RetroCraft's Asset Manager are hosted on CDN URLs. These are embedded directly in your exported HTML. The files will remain accessible as long as your RetroCraft account is active. For permanent hosting, consider downloading and re-hosting images on your own server.
