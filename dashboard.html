2. The Dashboard

When you open RetroCraft, you land on the Dashboard — your home base for managing all your website projects.

What You'll See

  • Project cards — each shows your site name, theme, and last edited date
  • [+ New Site] button — creates a brand new project
  • [Assets] navigation — jump to the Asset Manager to upload files

Project Card Actions

Each project card has three actions:

  • Open — loads the project into the editor
  • Duplicate — creates a copy of the entire project
  • Delete — permanently removes the project
