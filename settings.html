15. ⚙️Project Settings

Access project-wide settings from the Settings page.

Site Information

  • Site Title — appears in the browser tab and page titles
  • Tagline — short description shown in some themes
  • Author — your name or handle

Homepage

Choose which page serves as your homepage (exported as index.html).

GitHub Configuration

  • Personal Access Token — for Level 2 publishing (masked input)
  • Repository Name — the GitHub repo to publish to
  • [Connect & Test] — validates the token and checks repo status

Auto-Save

Your project auto-saves every 30 seconds. A "Saved ✓" indicator appears in the top bar. You can also save manually at any time with the [Save] button. Remember to hit the save button before you preview your page to capture any changes missed by auto save.

Undo / Redo

RetroCraft keeps at least 50 steps of history. Use the [Undo] and [Redo] buttons in the top bar, or keyboard shortcuts:
  • Ctrl+Z (Windows) / Cmd+Z (Mac) — Undo
  • Ctrl+Shift+Z (Windows) / Cmd+Shift+Z (Mac) — Redo
