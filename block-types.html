7. 🖼️Block Types Reference

Text

Rich text content. Edit directly on the canvas or via the sidebar. Supports bold, italic, links, and text alignment.

Heading

Large section titles. Choose H1, H2, or H3. Optional rainbow text effect for extra nostalgia.

Image

Display an image from two sources:
  • From Asset Library — pick from your uploaded assets or built-in stickers
  • From URL — paste any external image URL
Sizing options: Original Size, Fit Width, or Custom dimensions. Add alt text for accessibility and a link URL to make the image clickable.

Divider

Visual separators between sections. Styles: horizontal rule, animated GIF divider, rainbow line, or pixel dotted.

Button

A styled clickable button. Link type can be URL (any web link) or Email (opens the visitor's mail app with optional pre-filled subject). Button styles: default, bevel, sparkle, or terminal.

Link

A styled hyperlink that can display as text or an image. Link to another page in your site (internal) or any external URL. Text styles: default, button, retro underline, or blinking.

Marquee

Scrolling text — the classic early-web effect. Set direction (left/right), speed, and toggle a rainbow background. Pure CSS in the export — no JavaScript needed.

HTML Snippet

For advanced users. Paste raw HTML that renders in preview and export. Use this for embeds, custom widgets, or anything RetroCraft doesn't have a dedicated block for.

Gallery

A grid of images. Add images from your asset library or by URL. Choose 2, 3, or 4 columns. Toggle borders on or off.

Badge Row

A horizontal row of 88×31 badges (the classic web button size). Add badges from the built-in library or by URL. Set alignment and spacing.

Music Player

Add music to your site from two sources:
  • Audio File — upload an MP3/WAV via the Asset Manager
  • YouTube — paste any YouTube URL (full link, short link, or video ID)
Player styles include Winamp classic, cassette tape, and minimal retro bar. Toggle autoplay and loop.

Visitor Counter

Display a retro hit counter showing how many people have visited your page — the classic early-web badge of honor.

Heading (optional): Text shown above the counter, like "You are visitor number:"

Heading Alignment: Left, center, or right

Counter Style: Choose from five retro digit styles:

Digital — clean monospace LCD digits

Odometer — classic mechanical counter digits

LED — red glowing LED segments on black

Retro — green monospace on black (terminal style)

Simple — clean sans-serif digits

The counter works in the exported site without any JavaScript — it uses an image-based counter service that tracks visits server-side. Each counter has a unique ID so different pages get independent counts. No sign-up or account required.
⭐!TIP YouTube embeds use a standard <iframe> tag — no extra JavaScript on your site. The player loads from YouTube's servers, so it works on any static host including GitHub Pages.
