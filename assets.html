8. 📱Asset Management

The Asset Manager is your library for all images, GIFs, audio files, and decorative stickers used across your projects.

Accessing the Asset Manager

  • From the Dashboard — click "Assets" in the top navigation
  • From the Editor — click "Open Full Asset Manager" in the left sidebar Assets tab

Uploading Assets

  1. Open the Asset Manager
  2. Drag and drop files onto the upload zone, or click to browse
  3. Accepted formats: .jpg .png .gif .webp .mp3 .wav
  4. Uploaded files appear in your library immediately

My Assets Library

View all your uploaded assets in a grid. Each asset shows a thumbnail, filename, file type badge, and size. Actions per asset:
  • Copy URL — copies the CDN URL to clipboard
  • Insert into Page — adds the asset as an Image block on the current page
  • Delete — removes the asset permanently
Filter by type using the bar at the top: All / Images / GIFs / Audio / Badges.

Built-in Stickers & Badges

A curated collection of retro decorative assets is included with RetroCraft:
  • Blinkies — animated text badges
  • Badges — classic 88×31 web buttons
  • Dividers — animated separator GIFs
  • Sparkles — decorative glitter animations
  • Animals — cute pixel art creatures
These are read-only — click any to copy its URL or insert it into your page.
