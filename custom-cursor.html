10. Custom Cursors

Add a personal touch with a custom mouse cursor. Located in the Theme Customizer.

Preset Cursors

Six built-in options designed to match the nostalgic themes:

  • 🐱 Cat
  • 💕 Heart
  • ⭐ Star
  • 💀 Skull
  • 🗡️ Dagger
  • 💖 Pixel Heart

Custom Cursor from Asset Library

Toggle "Use custom cursor from Asset Library" to pick any image from your uploaded assets or by URL.

!NOTE Custom cursor images should be 32×32 pixels or smaller for full browser support. Chrome and Firefox limit cursor image dimensions — larger images may not display correctly. Resize your cursor image in any image editor before uploading.
!TIP Custom cursors use pure CSS (cursor: url('image.png'), auto;) — no JavaScript needed in your exported site.
